# Mon Portfolio Professionnel 💻✨

Bienvenue sur le dépôt GitHub de mon portfolio en ligne. Ce site vitrine me permet de présenter mon parcours, mes compétences de développeuse et les projets majeurs réalisés en formation et en entreprise.

🌐 **Lien du site en ligne :** https://nawel-benbachir.fr

## 🛠️ Stack technique & Outils
- **Structure & Style :** HTML5, CSS3, SCSS (Sass)
- **Dynamisme :** JavaScript
- **Backend :** PHP (Script de traitement sécurisé pour le formulaire de contact `send_email.php`)
- **Déploiement :** Hébergé sur o2switch avec gestion via cPanel, nom de domaine configuré chez IONOS

## 📁 Contenu & Pages clés
- **Accueil / Présentation :** Synthèse de mon profil et de mes aspirations.
- **Section Projets :** Focus sur mes réalisations concrètes :
  - `wingestion.html` : Présentation de mon projet de facturation en entreprise.
  - `webgestion.html` : Présentation de mon application de gestion Web (Laravel).
- **Formulaire de contact :** Entièrement fonctionnel grâce au script PHP personnalisé.
